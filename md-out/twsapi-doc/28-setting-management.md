## Setting Management

Beginning with TWS API 10.44, the API can be used to modify Trader Workstation settings as they relate to Orders, Precautions, and API Settings.  
While settings may be modified through this method, Read-Only access to the system must be first disabled manually through the GUI. See [TWS Settings](04.01-tws-config-api.md).
