### Trading The Overnight Session

In the event a user would like to designate a trade to take place during the Overnight trading hours, the [Order object](../twsapi-ref/23-order-ref.md) must set “includeOvernight” set to True and optionally set the Exchange value to OVERNIGHT.

#### 27.07.03 Trading The Overnight Session

Users that would like to route orders to [Overnight](/en/trading/ordertypes.php?m=overnightTradingModal) without trading during the regular session must set the [Order Object’s](../twsapi-ref/23-order-ref.md) includeOvernight value as True and designate the exchange value as “OVERNIGHT”.

order = Order()
    order.exchange = "OVERNIGHT"
    order.includeOvernight = True

#### Routing as Overnight+DAY

Users that would like to route orders to [Overnight+DAY](/en/trading/ordertypes.php?m=overnightSmartModal) to trade during the day and the overnight session must set the [Order Object’s](../twsapi-ref/23-order-ref.md) includeOvernight value as True and designate the exchange value as “SMART”.

order = Order()
    order.exchange = "SMART"
    order.includeOvernight = True
